== Description ==

This plugin adds a country flags widget to the [Elementor Page Builder](https://wordpress.org/plugins/elementor/). Elementor plugin must be activated to use this plugin. After you activate the required plugins, the widget will be available for use in Elementor.

*This plugin uses [flag-icon-css](https://github.com/lipis/flag-icon-css), A collection of all country flags in SVG â€” plus the CSS for easier integration.*

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Add New
4. Press the 'Edit with Elementor' button.
5. Now you can search for "Language Switcher" in the left panel.
6. Next, drag and drop the widget from the left panel onto the content area.
7. Build something amazing!
